Design a Student Portal interface for a school scheduling web application called "Smart Jadwal".

The system has two different portals in the same project:

/guru → for teachers to manage schedules, classes, and subjects

/siswa → for students to view their class schedule

The student login system is simple. Students do not use email or password.
Instead, they log in using:

Student Name

Class

The class options come from the classes created by the teacher/admin in the teacher portal.

Pages to Design
1. Student Login Page

A simple and clean login page for students.

Elements:

School logo at the top

Title: "Smart Jadwal - Student Portal"

Input field: Student Name

Dropdown: Select Class

Login button

Design style:

Minimal

Friendly for students

Blue and white color palette

2. Student Dashboard

After logging in, students see their dashboard.

Elements:

Top navigation bar

Welcome message: "Welcome, [Student Name]"

Information cards:

Class Name

Today's Schedule

Total Subjects

3. Weekly Schedule Page

A visual weekly timetable layout.

Layout:

Days displayed horizontally (Monday – Friday)

Time slots vertically

Each schedule block shows:

Subject name

Teacher name

Classroom

Time

4. Subjects Page

A simple list of subjects for the student's class.

Information displayed:

Subject name

Teacher

Schedule day

Design Style

Modern school dashboard

Clean UI

Rounded cards

Simple icons

Blue and white theme

Easy to read schedule layout

The design should look like a modern digital school timetable system for students.
